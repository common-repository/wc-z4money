O Plugin oficial Z4Money para WooCommerce.

== Description ==

= Em Português Brasil =
O plugin Z4Money para WooCommerce permite que lojistas aceitem pagamentos por boleto e cartão de crédito usando a Z4Money no WooCommerce.

= in English =
The Z4Money plugin for WooCommerce allows merchants to accept payment by boleto and credit card using Z4Money on WooCommerce.

== Installation ==

1. Acesse o painel de administrador do WordPress.
2. Na barra lateral direita, vá para: Plugins > Adicionar novo.
3. No campo de buscas entre com "Z4Money para WooCommerce". 
4. Selecione nosso plugin na lista e clique em "Instalar agora".
5. Por fim, clique no botão "Ativar" após a instalação.

= Manual Installation =

O método de instalação manual envolve o download do nosso plugin e o seu upload para o seu servidor web através do seu aplicativo FTP favorito. O codex do WordPress contém [instruções sobre como fazer isso aqui](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).

== Requirements ==

- Uma conta Z4Money
- Credenciais de acesso para API. [Entre em contato com a equipe Z4Money para obtê-las.](https://www.z4money.com.br/contato/ "Credenciais Z4Money")
- Um certificao SSL (Recomendado)
- Plugin "WooCommerce" ativado e instalado.
- Versão mínima do PHP  7.0

== Frequently Asked Questions ==

> O nosso FAQ é sempre atualizado com as dúvidas mais recebidas. Então fique atendo as nossas atualizações frequentes.

= Qual país a Z4Money atende? =
Atualmente a Z4Money recebe pagamento somente e exclusivamente do Brasil.

Se você tiver problemas, pode pedir ajuda no fórum de plugins.

= Tenho que fazer backup antes? =
Não necessariamente! Como qualquer desenvolvedor ou usuário do wordpress, criamos a mania de segurança e antes de instalar qualquer plugin faça o backup do banco de dados e arquivos.
